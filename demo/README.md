# Captain Misfortune by [Spritulus](https://www.spritulus.com/)
      
*Installation*

 1. Ensure you have Node.js v22 or higher and npm installed on your machine.

 2. Open a terminal and navigate to this directory.

 3. Run `npm install` to install the necessary dependencies.


*Usage*

Run `npm run dev` to see [Captain Misfortune](https://www.spritulus.com/sprites/44) in action.
